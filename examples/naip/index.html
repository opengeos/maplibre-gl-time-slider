USDA NAIP Annual Imagery (2009-2023)

This example displays annual NAIP (National Agriculture Imagery Program) imagery from the USDA.

The imagery uses false color composite (NIR, Red, Green) to highlight vegetation.

Use the Time Slider control to navigate through different years.

Click Add Layer to persist the current year as a permanent layer for comparison.

Dataset: USDA/NAIP/DOQQ
